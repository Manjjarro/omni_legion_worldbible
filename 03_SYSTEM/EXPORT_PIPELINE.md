# Export Pipeline

## Final deliverables
- `04_EXPORT/scenes/` for scene-level packages
- `04_EXPORT/winners/` for selected best runs
- `04_EXPORT/image_final/` for final stills
- `04_EXPORT/video_final/` for final edits

## Canonical selection rule
Only approved winners are allowed into final export folders.

## Suggested flow
1. Generate image runs
2. Review and score
3. Promote best image
4. Generate matching video runs
5. Review and score
6. Export winners to final folders
