# Prompt Engine

This file is kept as a legacy reference.

**Primary runtime moved to Cursor.**
Use `.cursor/rules/grimverse_shot_compiler.mdc` to compile generator-ready image prompts.

## What the compiled prompt should contain
- character visual anchors
- state-derived visual traits
- scene visual anchor
- camera
- composition
- style keywords

## What it should not contain
- world exposition
- event history
- story summary
- review commentary
- internal canon reasoning

Use Obsidian for canon.
Use Cursor for compilation.
