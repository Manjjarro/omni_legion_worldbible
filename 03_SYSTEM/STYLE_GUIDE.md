# Style Guide

## Visual Identity
Grimverse uses a hard-edged dark anime / cell-noir look.

## Do
- Keep figures readable against the background
- Use strong rim light or silhouette separation
- Keep action frames clean and intentional
- Keep wardrobe stable across a character arc

## Do Not
- Mix in unrelated styles
- Drift into heroic bright fantasy
- Use loose visual language in prompts
- Repeat the same descriptor unnaturally

## Prompt Rule
Always inject:
1. Character identity anchors
2. Character psychological anchors
3. Scene environment anchors
4. Shot camera and motion anchors
5. World rules
6. Style lock
