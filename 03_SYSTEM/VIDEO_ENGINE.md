# Video Engine

This file is kept as a legacy reference.

**Primary runtime moved to Cursor.**
Use `.cursor/rules/grimverse_shot_compiler.mdc` or `.cursor/rules/grimverse_thumbnail.mdc` for package generation.

## Final video prompt should contain
- subject
- environment
- camera movement
- motion strength
- lighting
- style keywords

## It should not contain
- world history
- story summary
- event chains
- review reasoning
- canon explanation
