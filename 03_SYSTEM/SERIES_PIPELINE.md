# Series Pipeline

## Recommended release structure
- Shorts for discovery
- 3–6 minute episodes for retention
- Compilations for session time
- Season cut for authority

## Winning logic
- Let the protagonist win some fights
- Let the protagonist lose important things
- Keep one major consequence per episode
- End on a question, reveal, or threat

## Audience model
The audience should root for competence first, morality second.
