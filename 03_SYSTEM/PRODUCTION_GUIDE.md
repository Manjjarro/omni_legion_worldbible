# Production Guide

## Canonical build order
1. Define arc
2. Define episode
3. Define scenes
4. Define shots
5. Generate image runs
6. Review and select winners
7. Generate video runs
8. Export final sequence

## Quality thresholds
- Character consistency: 9/10 or higher
- World logic: 9/10 or higher
- Style match: 9/10 or higher
- Motion: 8/10 or higher for video
- Final promo-worthy composite: 9/10 or higher
