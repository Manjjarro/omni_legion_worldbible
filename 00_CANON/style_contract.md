# STYLE CONTRACT (ABSOLUTE)

All generated visual outputs must satisfy:

## Rendering Identity
- 2D anime illustration style
- sharp ink linework
- flat color blocking
- high contrast lighting
- no photorealism
- no 3D shading
- no soft gradients

## Hard Failure Conditions (INVALID OUTPUT)
- realistic skin shaders
- volumetric lighting
- painterly brush textures
- cinematic blur depth of field
- CGI rendering style

## Consistency Rule
If style drifts, prompts must be revised BEFORE regenerating assets.

## Enforcement Principle
Style consistency > visual detail
