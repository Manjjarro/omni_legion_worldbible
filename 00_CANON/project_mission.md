# PROJECT MISSION

Goal:

Create a visually consistent AI-generated animated series.

Primary Objectives:

1. Maintain visual consistency.
2. Maintain lore consistency.
3. Build reusable asset library.
4. Improve prompts through iterative learning.
5. Preserve all successful discoveries.

Decision Hierarchy:

Canon >

Generated Assets

If conflicts occur, higher layers always win.
